Coding process!

Coding this whole website might've been exhausting, but with some simple principles, I was able to pull through.
Here are my no. rules when coding a new page:

  1. Copy and paste the 'genstruc.hmtl' structure. This is the basic code and strucutre every page of the website has to have.
  2. Use the 'base.html' or 'contact.html' page as an inspiration if I want to use columns
  3. Ask Wooper for design or content ideas
  4. Use W3Schools if I don't know how to code or format something
  5. Visit Stack Overflow if any issues occur
  6. Copy some cool code snippets from Codepen if I want to make my website stand out
  7. Last but not least, ask my dear, loyal friend Chat for some help if I could not get further (only occured twice ok?)
